# Module 12 – Azure DevOps Projects Based

Azure DevOps provides an end-to-end platform for managing code, pipelines, deployments, and infrastructure in modern DevOps workflows.

In real-world environments, teams use Azure DevOps to automate build, test, and deployment pipelines, manage microservices, and deploy applications to cloud platforms like App Service, AKS, and ACR.

This module focuses on:

- Understanding Azure DevOps ecosystem
- Building CI/CD pipelines using YAML and Classic methods
- Deploying applications to Azure App Service
- Working with containerized applications using ACR and AKS
- Managing microservices deployments
- Implementing full-stack CI/CD workflows
- Handling real-world Azure DevOps projects

This is a project-heavy module designed to simulate real enterprise pipelines.

## What You Will Learn

- Azure DevOps services (Repos, Pipelines, Boards, Artifacts)
- YAML pipelines vs Classic pipelines
- Building CI/CD pipelines for Node.js and Java applications
- Deploying applications to Azure App Service
- Using Azure Container Registry (ACR)
- Deploying applications to Azure Kubernetes Service (AKS)
- Managing microservices deployments using pipelines
- End-to-end full-stack CI/CD workflows
- Debugging pipeline failures
- Best practices for pipeline design

## How to Complete This Module

- Complete all tasks listed in tasks.md
- Create and run Azure DevOps pipelines
- Deploy applications to Azure services
- Work on both YAML and Classic pipelines
- Build and deploy containerized applications
- Execute real-world microservices deployment projects
- Capture logs, pipeline outputs, and issues
- Push your code to repositories
- Share your learning publicly

Pipeline failures and deployment issues are common.  
Understanding logs and fixing pipelines is a key DevOps skill.

## Outcome of This Module

After completing this module, you will be able to:

- Build CI/CD pipelines using Azure DevOps
- Work with both YAML and Classic pipelines
- Deploy applications to Azure App Service
- Use ACR and AKS for container-based deployments
- Implement microservices CI/CD pipelines
- Troubleshoot pipeline and deployment issues
- Design production-ready CI/CD workflows
